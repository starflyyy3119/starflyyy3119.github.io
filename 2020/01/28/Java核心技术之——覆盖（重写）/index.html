方法的签名

方法的 名字 和 参数列表 称为 方法的签名。

方法的覆盖（重写）

如果在子类中定义了一个与父类 签名 相同的方法，那么子类中的这个方法就覆盖了父类中的这个相同签名的方法。

返回类型 不是签名的一部分，在覆盖方法时，要保证返回类型的兼容性。（允许子类将覆盖方法的返回类型定义为原返回类型的子类型）
