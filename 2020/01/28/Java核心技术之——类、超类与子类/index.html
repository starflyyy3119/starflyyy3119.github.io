Java核心技术之——类、超类与子类

Posted on 2020-01-28 | In Java

超类与子类

已经存在的类称为 超类(superclass)、基类(base class)、父类(parent class); 新类称为 子类(subclass)、派生类(derived class) 或者 孩子类(child class)。

子类访问父类的特点

子类能够继承父类 所有的内容（包括私有变量，私有方法），但无法直接访问私有内容（拥有但无法访问），如果一定要访问，需要借助公共的接口。

子父类出现后，类成员的特点

变量

结果: 4 4 5
分析: 输出 z.num，实际上会输出 this.num，也就是当前类(子类)中定义的 num。想要调用父类中的 num，需要加上 super。

注意：

  • super 不是一个对象的引用，不能将 super 赋给另一个对象变量，它只是一个指示编译器调用超类方法的的特殊关键字。（Java核心技术卷一P152）

总结：

  • 如果子类中出现非私有的同名成员变量时，子类要访问本类中的变量，用 this; 子类要访问父类中的同名变量，用 super。

结果：4 4
分析：this 是子类对象的引用，子类继承了父类的所有变量，对于非父类私有的变量，可以直接访问。
