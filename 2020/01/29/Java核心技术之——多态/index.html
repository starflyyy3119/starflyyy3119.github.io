Java核心技术之——多态

Posted on 2020-01-29 | In Java

多态的概念

  • 一个对象变量可以指示多种实际类型的现象被称为多态(polymorphism)。

多态的体现

  • 父类的引用指向了自己的子类对象。
  • 父类的应用也可以接收自己的子类对象。

结果：eat fish eat dog
分析：对象方法的执行过程

多态的前提

  • 必须是类与类之间有关系，继承或者实现。
  • 通常还有一个前提，存在覆盖。

多态的好处

  • 提高了程序的扩展性。

多态的弊端

  • 只能使用父类的引用访问父类中有的成员。
